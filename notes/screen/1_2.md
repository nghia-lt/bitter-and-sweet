# Màn Hình 1.2 — Nhập Danh Sách Thành Viên

## UI Prompt

Design a mobile screen for entering player names in a party punishment game app called **"VÒNG QUAY TỚI SỐ"**.

### Style & Theme

- **Dark mode** with deep navy/black gradient background (#0A0A1A → #121228)
- **Neon accent colors**: electric purple (#A855F7), hot pink (#EC4899), cyan (#06B6D4)
- Subtle glow effects and neon borders
- Modern, sleek typography — uppercase for headers, clean sans-serif font
- Compact mobile-first layout (iPhone-style aspect ratio)
- No device frame — show only the UI itself

### Layout (Top → Bottom)

**① Header (top)**

- Small app logo + name "VÒNG QUAY TỚI SỐ" (compact, top-left)
- Back arrow icon (←) at top-left for navigation
- Step indicator: `BƯỚC 1/3` — small muted text, top-right

**② Title Section**

- Section title: **"👥 HỘI BẠN THÂN HÔM NAY GỒM AI?"** — bold, white text with subtle purple glow
- Subtitle: _"Thêm ít nhất 3 thành viên để bắt đầu"_ — smaller, muted gray text

**③ Input Area (middle)**

- A large text input field with neon purple border glow:
    - Placeholder text: _"Nhập tên thành viên..."_ — muted gray/purple
    - Dark glass-morphism background (semi-transparent dark with blur)
    - Rounded corners (border-radius: 12px)
- **➕ THÊM** button right next to the input — small pill button, filled purple gradient

**④ Member List (scrollable area, main content)**

- Each added member displays as a **card/chip** in a vertical list:
    - Dark card with subtle neon purple border (glass-morphism style)
    - Left side: random fun emoji avatar (😎🦋🤡💀🤖👻🦊🐸)
    - Center: member name in white bold text
    - Right side: ❌ delete icon button (muted red/pink, small)
- Cards have subtle entrance animation feel
- Example members shown:
    ```
    [ 😎 Nghĩa                    ❌ ]
    [ 🦋 Yến Nhi                  ❌ ]
    [ 🤡 Minh                     ❌ ]
    [ 💀 Hùng                     ❌ ]
    ```
- If less than 3 members: show a subtle warning text below the list in amber/yellow:
    > _"⚠️ Cần thêm ít nhất 3 thành viên"_

**⑤ Footer (bottom, fixed)**

- Member count badge: `4 THÀNH VIÊN` — small, muted purple text, center
- **TIẾP THEO ➡️** button — large pill-shaped, filled purple-to-pink gradient, white bold text, neon glow
    - If < 3 members: button is dimmed/disabled (gray, no glow)
    - If ≥ 3 members: button is active with full glow effect

### Key Visual Details

- Cards stack vertically with 8-12px gap
- Subtle purple glow on focused input field
- Rounded corners everywhere (12-16px)
- Dark glass-morphism panels with backdrop blur
- Smooth, modern layout with generous padding
- Portrait orientation, mobile viewport
