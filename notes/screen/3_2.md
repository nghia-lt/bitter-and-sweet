# Màn Hình 3.2 — Nhật Ký / Lịch Sử (Sổ Đen 📓)

## UI Prompt

Design a mobile screen for the **game history log** — a "black book" of all past punishments in the party game app **"VÒNG QUAY TỚI SỐ"**.

### Style & Theme

- **Dark mode** with deep navy/black gradient background (#0A0A1A → #121228)
- **Neon accent colors**: electric purple (#A855F7), hot pink (#EC4899), cyan (#06B6D4)
- Subtle glow effects and neon borders
- Modern, sleek typography — uppercase for headers, clean sans-serif font
- Compact mobile-first layout (iPhone-style aspect ratio)
- No device frame — show only the UI itself

### Layout (Top → Bottom)

**① Header (top)**

- Back arrow icon (←) at top-left
- Title: **"📓 SỔ ĐEN"** — bold, white text with subtle red/purple glow
- Subtitle: _"Ai Nợ Ai, Ghi Hết!"_ — muted red/gray, italic

**② Tab Bar (below header)**

- Two tabs, pill-style toggle:

    | Tab      | Label                                            |
    | -------- | ------------------------------------------------ |
    | Active   | 📜 **NHẬT KÝ** — shows chronological log         |
    | Inactive | 📊 **THỐNG KÊ** — shows player stats leaderboard |

- Active tab: filled purple gradient, white text
- Inactive tab: transparent, muted gray text, subtle border

---

### TAB 1: 📜 NHẬT KÝ (Chronological Log)

**③ Session Groups (scrollable)**

- Grouped by session date/time, newest first
- Each session has a **header bar**:

    ```
    📅 Phiên chơi: 03/03/2026, 20:30
    ```

    - Dark glass card, amber/yellow accent left-border, muted text

- Inside each session, **round cards** stacked vertically:

    **Round Card layout:**

    ```
    ┌──────────────────────────────────────────┐
    │  LƯỢT 1                                  │
    │  💀 Yến Nhi 🦋                           │
    │                                          │
    │  ├─ 🌶️ Ăn ớt (Tự chịu)                  │
    │  ├─ 💋 Hôn tay (Cùng Hùng)               │
    │  └─ 🎭 😩 Cam Chịu                       │
    └──────────────────────────────────────────┘
    ```

    - Dark glass-morphism card, subtle purple border
    - Victim name + avatar prominent at top, with 💀 skull
    - Penalty list as tree-style items with ├─ └─ connectors
    - Fate badge at bottom: color-coded per result type
        - 😩 Cam Chịu = gray
        - 💀 Chết Chùm = cyan (+ shows co-victim name)
        - 🏃 Thoát Kíp = green (+ shows who died instead)
        - ✨ Kim Thiền = golden (+ shows scapegoat name)

    **Special round example — Kim Thiền:**

    ```
    ┌──────────────────────────────────────────┐
    │  LƯỢT 2                                  │
    │  💀 Minh 🤡                              │
    │                                          │
    │  ├─ 🧄 Ăn tỏi sống (Tự chịu)            │
    │  ├─ ✋ Tét tay (Cùng Nghĩa)              │
    │  └─ 🎭 ✨ Kim Thiền → Hùng chết thay    │
    │                                          │
    │  ┌─ 💀 THẾ MẠNG: Hùng 💀 ──────────┐   │
    │  │  Nhận toàn bộ hình phạt từ Minh   │   │
    │  │  🎭 😩 Cam Chịu                  │   │
    │  └───────────────────────────────────┘   │
    └──────────────────────────────────────────┘
    ```

    - Nested sub-card for the scapegoat, cyan/purple border

### TAB 2: 📊 THỐNG KÊ (Player Stats Leaderboard)

**④ Player Leaderboard**

- Title: **"📊 BẢNG THÀNH TÍCH (THẤT BẠI)"** — purple text, bold
- Sorted by number of punishments (most = top), like a shame leaderboard

- Each player row as a card:

    ```
    ┌──────────────────────────────────────────┐
    │  🥇 1.  🦋 Yến Nhi                       │
    │                                          │
    │  Số lần bị phạt:  5                      │
    │  Nặng nhất:       🎲 Mix 3 (3 hình phạt) │
    │  Lần gần nhất:    03/03/2026             │
    └──────────────────────────────────────────┘
    ```

    - Rank badge: 🥇 gold (1st), 🥈 silver (2nd), 🥉 bronze (3rd), then numbers
    - Player avatar + name prominent
    - Stats in clean label: value pairs
    - Card border color intensity reflects rank (1st = hot pink glow, last = muted)

- Example leaderboard:
    ```
    🥇 Yến Nhi 🦋   — 5 lần — Mix 3
    🥈 Hùng 💀      — 4 lần — Kim Thiền → Thế mạng
    🥉 Minh 🤡      — 3 lần — Ăn ớt + Hôn má
    4. Nghĩa 😎     — 2 lần — Squat 10 cái
    ```

**⑤ Footer Info (bottom)**

- Small muted text: _"💾 Dữ liệu tự động lưu trên thiết bị. Bấm Reset ở trang chủ để xóa."_
- Muted gray/purple, small font

### Key Visual Details

- Tab switching is smooth with slide animation
- Session cards are collapsible/expandable (tap session header to fold)
- Round cards within a session have subtle left-border color matching the fate result
- The leaderboard has a "hall of shame" vibe — playful but dramatic
- Empty state: ghost illustration + _"Chưa có ai bị phạt. Bắt đầu chơi đi!"_
- Rounded corners everywhere (12-16px)
- Dark glass-morphism panels with backdrop blur
- Portrait orientation, mobile viewport
