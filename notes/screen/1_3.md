# Màn Hình 1.3 — Chọn Nguyên Liệu & Hình Phạt

## UI Prompt

Design a mobile screen for selecting ingredients and penalties in a party punishment game app called **"VÒNG QUAY TỚI SỐ"**.

### Style & Theme

- **Dark mode** with deep navy/black gradient background (#0A0A1A → #121228)
- **Neon accent colors**: electric purple (#A855F7), hot pink (#EC4899), cyan (#06B6D4)
- Subtle glow effects and neon borders
- Modern, sleek typography — uppercase for headers, clean sans-serif font
- Compact mobile-first layout (iPhone-style aspect ratio)
- No device frame — show only the UI itself

### Layout (Top → Bottom)

**① Header (top)**

- Small app logo + name "VÒNG QUAY TỚI SỐ" (compact, top-left)
- Back arrow icon (←) at top-left for navigation
- Step indicator: `BƯỚC 2/3` — small muted text, top-right

**② Title Section**

- Section title: **"🧪 KHO VŨ KHÍ HÔM NAY"** — bold, white text with subtle purple glow
- Subtitle: _"Chọn nguyên liệu có sẵn, hệ thống tự lọc hình phạt phù hợp"_ — muted gray

---

**③ Ingredient Toggles (top content area)**

- Sub-header: **"NGUYÊN LIỆU"** — small uppercase, cyan/purple accent text
- Row of 5 toggle chips/pills laid out horizontally (wrap if needed):

    | Toggle | Icon | Label     |
    | ------ | ---- | --------- |
    | ON     | 🍋   | Chanh     |
    | ON     | 🌶️   | Ớt        |
    | OFF    | 🫚   | Gừng      |
    | ON     | 🧄   | Tỏi       |
    | OFF    | 🥒   | Mướp đắng |

- **ON state**: filled purple/pink gradient background, white text, neon glow border
- **OFF state**: dark transparent background, muted gray text, dashed border
- Each toggle is a rounded pill shape, tappable

---

**④ Penalty List (scrollable, main content area)**

- Organized by **collapsible group sections** (accordion-style), each with a group header:

**Group 1: 🍽️ ĂN UỐNG** _(tag: ĐƠN — auto-filtered by ingredients)_

- Expandable section with neon purple left-border accent
- Inside: vertical list of penalty cards, each card shows:
    - Left: emoji icon + penalty name (e.g. "🍋 1/8 quả chanh")
    - Right: slot count badge `×3` in a small rounded chip (cyan), showing number of wheel slots
    - Each card has a **checkbox/toggle** on the far right to enable/disable
- Greyed-out penalties for unselected ingredients (e.g., gừng items dimmed if gừng is OFF)
- Sample visible items:
    ```
    ☑ 🍋 1/8 quả chanh          ×3
    ☑ 🍋 1/4 quả chanh          ×2
    ☑ 🍋 1/2 quả chanh          ×1
    ☐ 🫚 1 lát gừng             ×3  (dimmed — gừng OFF)
    ☑ 🌶️ 1 lát ớt               ×3
    ☑ 🧄 1 lát tỏi              ×2
    ```

**Group 2: 🎲 MIX** _(tag: COMBO)_

- Purple/pink accent border
- Cards:
    ```
    ☑ 🎲 Mix 2                  ×2
    ☑ 🎲 Mix 3                  ×1
    ```
- Small note below: _"Chỉ mix các hình phạt Ăn Uống"_ — muted italic

**Group 3: 💥 THỂ XÁC** _(tag: ĐÔI — cần 1 người cùng)_

- Cyan accent border
- Cards: tét tay (1/2/3/5 cái), tét đùi (1/2 cái), búng tai (1/2/3 cái), bóp vai, nhìn nhau không cười
    ```
    ☑ ✋ Tét tay 1 cái           ×3
    ☑ ✋ Tét tay 2 cái           ×3
    ☑ 👂 Búng tai 1 cái         ×3
    ☑ 💆 Bóp vai 30 giây        ×2
    ☑ 😐 Nhìn nhau không cười   ×2
    ```

**Group 4: 💪 THỂ LỰC** _(tag: ĐƠN)_

- Green/cyan accent border
- Cards: squat, plank, đứng 1 chân, nhảy lò cò

**Group 5: 😘 NHẠY CẢM** _(tag: ĐÔI + NGOẠI LỆ)_

- Hot pink accent border, with a small ⚠️ warning icon
- Cards:
    ```
    ☑ 💋 Hôn tay                ×1
    ☑ 😘 Hôn má                 ×1
    ```
- Small warning text: _"Áp dụng quy tắc Ngoại Lệ ở bước tiếp theo"_ — amber text

**Group 6: 📱 XÃ HỘI** _(tag: ĐƠN)_

- Yellow/amber accent border
- Cards: đăng story, hát bài, nói "em yêu anh/chị"

---

**⑤ Footer (bottom, fixed)**

- Summary bar: `TỔNG: 52 Ô TRÊN VÒNG QUAY` — small, muted purple text with neon badge
- **TIẾP THEO ➡️** button — large pill-shaped, filled purple-to-pink gradient, white bold text, neon glow

### Key Visual Details

- Group headers are tappable to expand/collapse (accordion)
- Each group has a distinct colored left-border accent for quick visual identification
- Slot count badges (×3, ×2, ×1) use different intensity: ×3 = bright cyan, ×2 = medium, ×1 = dim/rare-feeling hot pink
- Smooth scroll with group headers that stick when scrolling
- Rounded corners everywhere (12-16px)
- Dark glass-morphism panels with backdrop blur
- Portrait orientation, mobile viewport
