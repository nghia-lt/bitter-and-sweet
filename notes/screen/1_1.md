# Màn Hình 1.1 — Màn Hình Chào Mừng

## UI Prompt

Design a mobile welcome screen for a party punishment game app called **"VÒNG QUAY TỚI SỐ"** (Spin Wheel of Fate).

### Style & Theme

- **Dark mode** with deep navy/black gradient background (#0A0A1A → #121228)
- **Neon accent colors**: electric purple (#A855F7), hot pink (#EC4899), cyan (#06B6D4)
- Subtle glow effects and neon borders
- Modern, sleek typography — uppercase for titles, clean sans-serif font
- Compact mobile-first layout (iPhone-style aspect ratio)

### Layout (Top → Bottom)

**① Header Area (top)**

- App logo: a stylized spinning wheel icon with purple/pink neon glow
- App name: **"VÒNG QUAY TỚI SỐ"** — bold uppercase, white text with subtle purple glow
- Tagline below: _"PENALTY PROTOCOL"_ — smaller, muted purple/gray text
- Settings gear icon (⚙️) at top-right corner, subtle gray

**② Center Area (middle — main focus)**

- A large, eye-catching illustration or animated preview of a neon spinning wheel — glowing purple/cyan rim, dark center
- Below the wheel preview, a dramatic tagline in neon text:
    > _"Thua là phải chịu. Không chạy đâu được."_
- Subtle floating particles or sparkle effects in the background

**③ Action Buttons (bottom section)**

- **Primary button**: `🎮 BẮT ĐẦU PHIÊN MỚI` — large, pill-shaped, filled with vibrant purple-to-pink gradient, white bold text, subtle glow/shadow
- **Secondary button**: `📖 XEM LỊCH SỬ` — same size, outlined style with neon purple border, transparent/dark fill, purple text

**④ Footer Info (very bottom, conditional)**

- If previous session data exists, show a small info banner:
    > _"⚡ Dữ liệu phiên trước vẫn còn!"_
    - Muted yellow/amber text, small font
- **🔄 Reset** — small text link at bottom-right, gray/muted color

### Key Visual Details

- All interactive elements have a subtle neon glow on hover/focus
- Rounded corners on all cards and buttons (border-radius: 12-16px)
- Dark glass-morphism panels (semi-transparent dark backgrounds with blur)
- No device frame — show only the UI itself
- Portrait orientation, mobile viewport
