# Màn Hình 2.1 — Lên Thớt (Chọn Nạn Nhân)

## UI Prompt

Design a mobile screen for selecting the losing player (victim) in a party punishment game app called **"VÒNG QUAY TỚI SỐ"**.

### Style & Theme

- **Dark mode** with deep navy/black gradient background (#0A0A1A → #121228)
- **Neon accent colors**: electric purple (#A855F7), hot pink (#EC4899), cyan (#06B6D4)
- Subtle glow effects and neon borders
- Modern, sleek typography — uppercase for headers, clean sans-serif font
- Compact mobile-first layout (iPhone-style aspect ratio)
- No device frame — show only the UI itself

### Layout (Top → Bottom)

**① Header (top)**

- Small app logo + name "VÒNG QUAY TỚI SỐ" (compact, top-left)
- 📓 History icon (top-right) — small, muted, links to game log

**② Title Section**

- Section title: **"🔪 AI LÊN THỚT?"** — bold, large uppercase, white text with intense red/orange neon glow
- Dramatic subtitle: _"Chọn kẻ thua cuộc. Không có đường lui."_ — smaller, muted red/gray text
- Subtle animated knife emoji or slash effect near the title

---

**③ Player Grid (main content area)**

- All members displayed as **large tappable cards** in a **2-column grid** layout
- Each player card:
    - Dark glass-morphism background with subtle purple border
    - Large emoji avatar centered at the top (😎🦋🤡💀)
    - Player name below — white bold text, centered
    - **Penalty count badge** — small rounded chip at top-right corner of the card:
        - Shows "×0" / "×1" / "×3" etc. — number of times this person has been punished
        - Color: cyan for low count, orange for medium, red/hot pink for high count
    - On hover/press: card border glows brighter purple, slight scale-up effect

- Example grid (4 players):
    ```
    ┌──────────────┐  ┌──────────────┐
    │         ×1   │  │         ×3   │
    │     😎       │  │     🦋       │
    │   Nghĩa      │  │  Yến Nhi     │
    └──────────────┘  └──────────────┘
    ┌──────────────┐  ┌──────────────┐
    │         ×0   │  │         ×2   │
    │     🤡       │  │     💀       │
    └──────────────┘  └──────────────┘
    │    Minh       │  │    Hùng      │
    └──────────────┘  └──────────────┘
    ```

---

**④ Selected State (overlay/transition — after tapping a player)**

- When a player card is tapped, the card **zooms to center** of screen with a spotlight/radial light effect
- Background dims to near-black with subtle red vignette
- Selected player card becomes larger, with intense neon glow (red/pink border)
- Dramatic text appears below the card:
    > **"💀 [TÊN] ĐÃ BƯỚC LÊN ĐOẠN ĐẦU ĐÀI! 💀"**
    - Red/hot pink glowing text, bold uppercase
- Subtle skull particles or flame effects floating around the card

- **QUAY HÌNH PHẠT 🎰** button — large pill-shaped, filled red-to-purple gradient, white bold text, pulsing neon glow
- Small "Chọn lại" text link below — muted gray, to go back to grid

### Key Visual Details

- Grid cards have equal size, generous padding, 12-16px gap
- Cards feel "selectable" — shadow lift on press, subtle bounce animation
- The transition from grid → selected state should feel dramatic and theatrical
- Penalty count badges create a sense of history and "revenge" potential
- Rounded corners everywhere (12-16px)
- Dark glass-morphism panels with backdrop blur
- Portrait orientation, mobile viewport
