# Màn Hình 2.3 — Vòng Quay Nhân Phẩm (Cơ Hội Cuối)

## UI Prompt

Design a mobile screen for the **"Fate Wheel"** — a dramatic last-chance spin that determines the victim's final fate. This screen has multiple states. Party game app **"VÒNG QUAY TỚI SỐ"**.

### Style & Theme

- **Dark mode** with deep navy/black gradient background (#0A0A1A → #121228)
- **Neon accent colors**: electric purple (#A855F7), hot pink (#EC4899), cyan (#06B6D4)
- Subtle glow effects and neon borders
- Modern, sleek typography — uppercase for headers, clean sans-serif font
- Compact mobile-first layout (iPhone-style aspect ratio)
- No device frame — show only the UI itself
- **This screen should feel the MOST dramatic of all** — heavy use of glow, particles, tension

---

## STATE A: Vòng Quay Nhân Phẩm (Default — trước khi quay)

### Layout (Top → Bottom)

**① Dramatic Intro**

- Screen briefly fades to black, then slowly reveals with a dramatic glow
- Title appears with intense animation:
    > **"🎭 VÒNG QUAY NHÂN PHẨM"**
    - Large, bold uppercase, white text with golden/amber neon glow
- Subtitle:
    > _"Số phận chưa kết thúc ở đây đâu..."_
    - Muted gray, italic, mysterious vibe

**② Current Punishment Summary (compact card)**

- Small dark glass-morphism card showing what's already been decided:
    > **Hình phạt hiện tại cho [Tên]:**
    > 🧄 2 lát tỏi · 💋 Hôn tay Hùng
- Compact, one-line summary style, muted purple border

**③ Fate Wheel (center — hero element)**

- A **4-segment wheel** — each segment has distinctly different SIZE reflecting probability:

    | Segment                | Size                                       | Color                      | Label               |
    | ---------------------- | ------------------------------------------ | -------------------------- | ------------------- |
    | 😩 Cam Chịu            | **83%** — massive, takes most of the wheel | Dark gray/muted purple     | "Chấp nhận số phận" |
    | 💀 Chết Chùm           | **10%** — medium slice                     | Cyan/teal neon             | "Kéo thêm 1 người"  |
    | 🏃 Thoát Kíp           | **5%** — small slice                       | Green/emerald neon         | "Chọn lại nạn nhân" |
    | ✨ Kim Thiền Thoát Xác | **2%** — tiny sliver!                      | GOLDEN/YELLOW intense glow | "THẾ MẠNG"          |

- The wheel has a **neon rim** and glowing pointer at the top
- Center button: **"QUAY!"** — circular, pulsing golden gradient, dramatic feel
- Tiny text below wheel: percentage labels for each segment

**④ Tension Text (bottom)**

- _"Mỗi khi kim gần dừng ở ô hiếm... hãy hét lên."_ — muted gray, small

---

## STATE B: Kết Quả — 😩 Cam Chịu (83%)

- Wheel stops on the massive gray segment
- Screen displays with a defeated/somber tone:
    ```
    ┌──────────────────────────────────────┐
    │                                      │
    │         😩😩😩                       │
    │   NHÂN PHẨM HẾT RỒI BẠN ƠI          │
    │                                      │
    │   Giữ nguyên TOÀN BỘ hình phạt.     │
    │   Không thương lượng.                │
    │                                      │
    └──────────────────────────────────────┘
    ```
- Dark, muted card — no fireworks, no joy — just acceptance
- Button: **CHẤP NHẬN SỐ PHẬN ➡️** — muted purple pill button

---

## STATE C: Kết Quả — 💀 Chết Chùm (10%)

- Wheel stops on cyan segment — moderate surprise!
- Flash effect:

    > **"💀 CHẾT CHÙM! Kéo thêm 1 người! 💀"**
    - Cyan neon glow, skull particles

- **Player selection wheel** appears (smaller):
    - Contains all members except current victim
    - Player taps QUAY → selects the unlucky co-victim

- After selection, **split summary table** appears:
    ```
    ┌──────────────────────────────────────────┐
    │  💀 CHIA ĐỀU — CHIA ĐẸP                 │
    │                                          │
    │  Hình phạt gốc    │ Minh    │ Nghĩa     │
    │  ─────────────────┼─────────┼────────── │
    │  🧄 2 lát tỏi     │ 1 lát   │ 1 lát     │
    │  🍋 3 lát chanh   │ 1,5 lát │ 1,5 lát   │
    │  ✋ Tét tay 4 cái │ 2 cái   │ 2 cái     │
    │  😘 Hôn má Y.Nhi  │ Giữ ✅  │ Không ❌   │
    └──────────────────────────────────────────┘
    ```
- Split rules legend (small text below table):
    - 🍽️💥 Đơn/Đôi → Chia đôi | 💋 Hôn → Không chia
- Button: **TIẾP TỤC ➡️**

---

## STATE D: Kết Quả — 🏃 Thoát Kíp (5%)

- Wheel stops on green segment — excitement builds!
- Screen explosion:

    > **"🏃💨 THOÁT! THOÁT THẬT RỒI!"**
    > _"Nhưng ai đó phải chết thay..."_
    - Green neon glow, running/escape particles

- **Player selection wheel** appears:
    - Contains ALL members (including person who just escaped)
    - Player taps QUAY → selects the new victim

- If exclusion conflict detected → amber warning + auto re-roll partner:

    > _"⚠️ Xung đột ngoại lệ: Quay lại người thực hiện cùng cho hình phạt Hôn tay..."_

- Final result card:
    > **"🎯 [Tên mới] đã bị chọn! Tội nghiệp quá!"**
- Button: **TIẾP TỤC ➡️**

---

## STATE E: Kết Quả — ✨ Kim Thiền Thoát Xác (2%)

- Wheel stops on the TINY golden sliver — **LEGENDARY MOMENT!**
- **MASSIVE screen explosion**: fireworks, confetti, golden particles, screen flash white→gold

    > **"✨✨✨ HUYỀN THOẠI! ✨✨✨"**
    > **"KIM THIỀN THOÁT XÁC!"**
    - Golden/yellow neon glow, ultra dramatic, pulsing text
    - Firework particle effects fill the entire screen

- Subtitle:

    > _"[Tên] đã giác ngộ! Toàn bộ hình phạt CHUYỂN THẲNG cho người khác!"_

- **Player selection wheel** appears:
    - Contains all members EXCEPT the escaped victim
    - Golden-themed wheel (special feel)
    - Player taps QUAY → selects the scapegoat

- If exclusion conflict → same amber warning + re-roll as Thoát Kíp

- Final result card with dramatic glow:
    > **"💥 [Tên mới] nhận hết!"**
    > **"[Nạn nhân cũ] cười trên nỗi đau người khác! 💥"**
- Button: **TIẾP TỤC ➡️**

---

### Key Visual Details

- Each result state has a COMPLETELY different visual tone:
    - 😩 Cam Chịu = somber, gray, no effects
    - 💀 Chết Chùm = cyan flash, moderate drama
    - 🏃 Thoát Kíp = green flash, excitement
    - ✨ Kim Thiền = GOLDEN EXPLOSION, maximum drama
- The 4-segment wheel should make the probability visually obvious — 83% dominates, 2% is a barely visible sliver
- Split table in Chết Chùm uses clear 3-column layout with color-coded rules
- Rounded corners everywhere (12-16px)
- Dark glass-morphism panels with backdrop blur
- Portrait orientation, mobile viewport
