# Màn Hình 2.2 — Phán Quyết (Vòng Quay Hình Phạt)

## UI Prompt

Design a mobile screen with a **spinning punishment wheel** for a party game app called **"VÒNG QUAY TỚI SỐ"**. This is the core gameplay screen with multiple states.

### Style & Theme

- **Dark mode** with deep navy/black gradient background (#0A0A1A → #121228)
- **Neon accent colors**: electric purple (#A855F7), hot pink (#EC4899), cyan (#06B6D4)
- Subtle glow effects and neon borders
- Modern, sleek typography — uppercase for headers, clean sans-serif font
- Compact mobile-first layout (iPhone-style aspect ratio)
- No device frame — show only the UI itself

---

## STATE A: Vòng Quay Chính (Default)

### Layout (Top → Bottom)

**① Header (top)**

- Victim's name displayed prominently:
    > **"Số phận của [Yến Nhi 🦋] nằm ở đây... 🎯"**
    - White text, medium size, centered, subtle pulse animation
- 📓 History icon (top-right corner)

**② Main Spinning Wheel (center — hero element)**

- A large **circular wheel** taking up ~60% of the screen width
- The wheel is divided into **many colored segments** (like the reference image's vertical list, but as a circle):
    - Each segment = one penalty slot with its icon + short label
    - Segment colors vary by penalty group:
        - 🍽️ Ăn Uống: purple/violet shades
        - 💥 Thể Xác: cyan/teal shades
        - 💪 Thể Lực: green/emerald shades
        - 😘 Nhạy Cảm: hot pink shades
        - 📱 Xã Hội: amber/yellow shades
        - 🎲 Mix: red/orange shades (danger feel)
    - Segments have different widths based on slot count (×3 = wider, ×1 = narrow)
- **Neon glow rim** around the wheel — purple/cyan gradient
- **Pointer/Arrow** at the top of the wheel — bright red/pink triangle, glowing
- Center of the wheel: **"QUAY!"** button — circular, pulsing purple-to-pink gradient, white bold text

**③ Currently Highlighted Penalty (below wheel)**

- Similar to the reference image: a **focused card** showing the penalty the pointer is nearest to
- Dark glass-morphism card with bright neon border (matches penalty group color)
- Shows: group tag label (small text, e.g. "PENALTY 01"), icon + penalty name (large bold text)
- This card updates as the wheel spins
- Example:
    ```
    ┌────────────────────────────────┐
    │  CRITICAL FOCUS                │
    │  🍋 1/8 QUẢ CHANH             │
    │  ━━━━━━━━● ○ ○                │
    └────────────────────────────────┘
    ```

**④ Footer tagline**

- _"TARGET THE WINDOW AND UNLEASH CHAOS"_ style dramatic text — muted gray, small

## STATE B: Kết Quả Đơn (sau khi quay xong — hình phạt đơn)

- Wheel fades/shrinks to background
- **Result card** zooms in center with explosive neon effects:
    ```
    ┌────────────────────────────────────┐
    │                                    │
    │   🌶️🌶️🌶️                          │
    │   2 LÁT ỚT!                       │
    │                                    │
    │   "[Tên] ơi, cay lắm đó nha! 😈"  │
    │                                    │
    └────────────────────────────────────┘
    ```
- Card has intense glow matching penalty group color
- Confetti/particle effects around the card
- Button below: **ĐÃ RÕ! TIẾP TỤC ➡️** — purple gradient pill

---

## STATE C: Kết Quả Đôi (hình phạt cần thêm 1 người)

- Result card shows the penalty:

    > **"💋 HÔN TAY!"**
    > _"Ôi trời ơi! Nhưng hôn tay AI đây?!"_

- Below, a **secondary smaller wheel** appears with player names:
    - Each segment = one player (avatar emoji + name)
    - Excluded players (by ngoại lệ rules) are NOT on the wheel
    - If exclusion applied: small amber note below:
        > _"⚠️ Ngoại lệ: Nghĩa đã bị loại khỏi vòng quay"_
    - Center button: **"QUAY!"** — smaller but same style

- After spinning, final result card:
    ```
    ┌────────────────────────────────────┐
    │  💋 Yến Nhi phải HÔN TAY Hùng! 💋 │
    │  "Hùng ơi, đưa tay đây! 🤚😏"     │
    └────────────────────────────────────┘
    ```
- Button: **ĐÃ RÕ! TIẾP TỤC ➡️**

---

## STATE D: Kết Quả Mix (combo — quay thêm nhiều lần)

- Flash screen effect:

    > **"⚡ MIX 2 — COMBO TỬ THẦN! ⚡"**
    > _"Quay thêm 2 phát nữa nè!"_

- Wheel re-appears with **only food penalties** (filtered)
- Auto-spins or player taps QUAY for each sub-spin
- After all sub-spins complete, a **summary table card** appears:
    ```
    ┌──────────────────────────────────────┐
    │  📋 PHÁN QUYẾT CHO [TÊN]:           │
    │                                      │
    │  #1  🧄 2 lát tỏi        Tự chịu    │
    │  #2  🥒 1 miếng mướp đắng Tự chịu   │
    └──────────────────────────────────────┘
    ```
- Dark glass card with purple neon border, table rows inside
- Button: **ĐÃ RÕ! TIẾP TỤC ➡️**

---

### Key Visual Details

- The wheel is the star — colorful, vibrant, with depth and 3D-ish bevel effect
- Spinning animation: fast → decelerate → wobble → stop with tick-tick sounds feel
- Each state transition has dramatic flair (zoom, flash, particles)
- The "focused penalty" card below the wheel mimics the reference image's highlighted card style
- Rounded corners everywhere (12-16px)
- Dark glass-morphism panels with backdrop blur
- Portrait orientation, mobile viewport
