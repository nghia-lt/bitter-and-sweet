# Màn Hình 3.1 — Kết Quả Cuối Cùng

## UI Prompt

Design a mobile screen showing the **final verdict** — a dramatic sentencing screen for the party punishment game app **"VÒNG QUAY TỚI SỐ"**.

### Style & Theme

- **Dark mode** with deep navy/black gradient background (#0A0A1A → #121228)
- **Neon accent colors**: electric purple (#A855F7), hot pink (#EC4899), cyan (#06B6D4)
- Subtle glow effects and neon borders
- Modern, sleek typography — uppercase for headers, clean sans-serif font
- Compact mobile-first layout (iPhone-style aspect ratio)
- No device frame — show only the UI itself

### Layout (Top → Bottom)

**① Header (top)**

- Small app logo + name "VÒNG QUAY TỚI SỐ" (compact, top-left)
- 📓 History icon (top-right corner)

**② Title Section**

- Title: **"🏛️ PHÁN QUYẾT CUỐI CÙNG"** — large, bold uppercase, white text with red/crimson neon glow
- Subtitle:
    > _"Không kháng cáo. Không thương lượng. Chỉ có CHỊU."_
    - Muted red/gray, italic, dramatic

### Key Visual Details

- The main victim card feels like a "court verdict" / "sentencing document" — formal but dark
- Penalty table inside the card is clean and scannable
- If Chết Chùm: two cards shown side-by-side or stacked, clearly labeled "NẠN NHÂN" and "BẠN NẠN"
- The green "ĐÃ HOÀN THÀNH" button signals finality and transition to next round
- Rounded corners everywhere (12-16px)
- Dark glass-morphism panels with backdrop blur
- Portrait orientation, mobile viewport
