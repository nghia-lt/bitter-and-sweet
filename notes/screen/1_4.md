# Màn Hình 1.4 — Thiết Lập Ngoại Lệ

## UI Prompt

Design a mobile screen for setting up exclusion rules in a party punishment game app called **"VÒNG QUAY TỚI SỐ"**.

### Style & Theme

- **Dark mode** with deep navy/black gradient background (#0A0A1A → #121228)
- **Neon accent colors**: electric purple (#A855F7), hot pink (#EC4899), cyan (#06B6D4)
- Subtle glow effects and neon borders
- Modern, sleek typography — uppercase for headers, clean sans-serif font
- Compact mobile-first layout (iPhone-style aspect ratio)
- No device frame — show only the UI itself

### Layout (Top → Bottom)

**① Header (top)**

- Small app logo + name "VÒNG QUAY TỚI SỐ" (compact, top-left)
- Back arrow icon (←) at top-left for navigation
- Step indicator: `BƯỚC 3/3` — small muted text, top-right

**② Title Section**

- Section title: **"⚙️ LUẬT NGẦM — VÙNG CẤM ĐỊA"** — bold, white text with subtle red/pink glow
- Subtitle in italic, muted gray:
    > _"Có những điều... không nên xảy ra. Cài đặt ở đây để hệ thống tự hiểu."_
- A small 🤫 emoji or "SECRET" badge next to the title — hot pink accent

---

**③ Exclusion Rules List (main content area, scrollable)**

- Each rule is displayed as a **dark glass-morphism card** with hot pink/red left-border accent:

**Rule Card layout:**

```
┌─────────────────────────────────────────┐
│  QUY TẮC #1                         🗑️ │
│                                         │
│  Khi hình phạt là:                      │
│  ┌─────────────────────────────┐        │
│  │ 💋 Hôn tay, 😘 Hôn má    ▼ │        │
│  └─────────────────────────────┘        │
│                                         │
│  Thì hai người này:                     │
│  ┌────────────┐   ⚡   ┌────────────┐  │
│  │ 😎 Nghĩa ▼ │       │ 🦋 Yến Nhi▼│  │
│  └────────────┘        └────────────┘  │
│                                         │
│  → ❌ KHÔNG xuất hiện cùng nhau         │
│     (dù ai phạt ai)                    │
└─────────────────────────────────────────┘
```

- **Dropdown fields** styled as dark input boxes with neon purple border, dropdown arrow (▼)
    - Penalty dropdown: multi-select, shows selected items as tags/chips
    - Person dropdowns: single-select each, show member avatar emoji + name
- **⚡ icon** between the two person dropdowns — glowing electric effect
- **🗑️ delete button** at top-right of each rule card — muted red, small
- **"❌ KHÔNG xuất hiện cùng nhau"** — red/pink bold text at bottom of each card

**Example rule shown:**

- Rule #1: Hôn tay + Hôn má → Nghĩa ⚡ Yến Nhi → ❌ KHÔNG

---

**④ Add Rule Button (below rules list)**

- **➕ THÊM QUY TẮC** — outlined button, dashed neon purple border, icon + text
- Centered, full-width, tappable

---

**⑤ Info Note (small, above footer)**

- Small muted text with 💾 icon:
    > _"💾 Thiết lập tự động ghi nhớ trên thiết bị. Lần sau không cần nhập lại."_
- Muted gray/purple, italic, small font

---

**⑥ Footer (bottom, fixed)**

- **🚀 VÀO TRẬN!** button — extra large pill-shaped, filled with vibrant **red-to-purple gradient** (more dramatic than previous steps), white bold uppercase text, intense neon glow, pulsing animation effect
- This button feels more dramatic/final compared to "Tiếp theo" buttons on previous screens

### Key Visual Details

- Rule cards have a "classified/secret" vibe — subtle crosshatch or diagonal line pattern in the card background
- Hot pink / red accent theme for this screen (danger/forbidden feel), different from the purple of previous screens
- Dropdown fields glow on focus/expanded state
- Rounded corners everywhere (12-16px)
- Dark glass-morphism panels with backdrop blur
- Empty state (no rules yet): show a ghost illustration with text _"Chưa có quy tắc nào. Bấm '➕ Thêm Quy Tắc' để bắt đầu."_
- Portrait orientation, mobile viewport
